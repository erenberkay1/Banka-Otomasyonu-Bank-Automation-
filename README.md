## 🌟 Features / Özellikler

### [EN] English
- **Balance Management:** View and manage account balances.
- **Transactions:** Deposit and withdraw funds securely.
- **Money Transfer:** Perform EFT/Transfer between different accounts.
- **Data Persistence:** Uses SQLite to store all financial records.
- **API Testing:** Full Swagger support for testing endpoints.

### [TR] Türkçe
- **Bakiye Yönetimi:** Hesap bakiyelerini görüntüleme ve yönetme.
- **İşlemler:** Güvenli şekilde para yatırma ve çekme.
- **Para Transferi:** Hesaplar arası EFT/Havale işlemleri.
- **Kalıcı Veri:** Tüm finansal kayıtlar için SQLite kullanımı.
- **API Testi:** Endpoint'leri test etmek için tam Swagger desteği.

---

## 🛠️ Tech Stack / Teknolojiler
- **Backend:** C# .NET 8 Web API
- **Database:** SQLite
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
